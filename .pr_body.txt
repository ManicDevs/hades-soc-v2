This PR implements the following changes (automated, low-risk):

- Pure-Go kernel simulator: `os/hades-go-kernel` now includes a userspace kernel (`hades-kernel`) with an interactive shell and a small Supervisor for external services.

  - Health monitoring for services (HTTP health endpoint checks + restart).
  - Shell commands to manage services: `service start|stop|status|health|list`.
